- obsahují data o Entitách (ucelený předmět např. uživatel, kiha, třída, auto, ...)

- databázové systémy: např. **MariaDB**, *MySQL*, Oracle, MSSQL, **PostgreSQL**, **SQLite**
- Structured Query Language = SQL = Strukturovaný jazyk pro dotazy
